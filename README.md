#  Rainstalker

[![Minecraft Version](https://img.shields.io/badge/Minecraft-1.21-brightgreen.svg)](https://www.minecraft.net/)
[![Platform](https://img.shields.io/badge/Loader-Fabric-blue.svg)](https://fabricmc.net/)

This mod adds a new mob to the world; the Rainstalker. Those who overcome the evasive threat will unlock the path to a new dimension; the Downpour.

## 🛠️ Features

* **Rainstalker:** A mob that only spawns during the rain and stalks players.
* **The Downpour:** A dimension home to the rainstalker that must be accessed through special means.
* **Sodden Blocks:** The downpour dimension is forever wet, thus the natural blocks there have been soaked to their core.

## 📦 Installation

1.  Ensure you have the **Fabric Loader** installed for Minecraft 1.21.
2.  Download the latest release from the [Releases](https://github.com/rakstern/rainstalker-1.21/releases) page.
3.  Place the `.jar` file into your `%appdata%/.minecraft/mods` folder.
4.  Profit???
